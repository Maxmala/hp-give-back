Pourquoi ? Pour le jeu

Le vrai mot français est “programmer”.

Le codage, c’est la programmation informatique, ce qui permet l’écriture des programmes pour développer des logiciels, ou une page web, par exemple, mais aussi pour dire à une machine ce qu’elle doit faire, la guider.

Parce-que l’on aimerait que cette page d’infos s’ouvre lorsqu’on clique là, que le texte apparaisse sur fond rose, que la machine dirige le jet d’eau pression comme il faut pour découper une pièce de métal, etc … Les finalités peuvent être très diverses mais le code est le langage initial nécessaire à ces usages.

Pourquoi ? ou pour en vivre

Le monde du travail change et le numerique au sens large va prendre de + en + de place.

L'avantage avec l'informatique est que l'on peut évoluer vers differents metiers et/ou acquerir de l'independence vis à vis d'un univers particulier.

Savoir comment fonctionne le Web , internet, ce qu'il y a derière une jolie page web peut donc être intéréssant professionellement.

Fini les formations traditionelles dans une école Les serious games ou les jeux vidéos « sérieux » Une autre méthode d’apprentissage de la programmation consiste à le faire dans le cadre d’un jeu vidéo : coder, pour résoudre une énigme, obtenir des options supplémentaires, débloquer un niveau… C’est le cas de CodeCombat, un jeu français qui initie au JavaScript ou au Python en invitant le joueur à agir sur le code pour faire bouger des personnages, débloquer des situations ou jeter des sorts. Il y a aussi Gleamcode, qui sera proposé aux associations, aux entreprises, voire même à l’Education nationale, pour apprendre le code aux enfants via un univers ludique.

Certaines Limites à ces méthodes d'appretissage à distance : cela nécéssité de grande dicipline et autonomie sans proximité humaine. Les MOOC ont beaucoup de succés au début mais ne sont généralement pas suivient jusqu'a la fin ( seulement 10%)
